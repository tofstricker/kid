# Building KiteControl in the Cloud (No Android Studio)

If you don't want to install Android Studio or the Android SDK on your computer, you can use a **CI/CD Service** to build the APK in the cloud.

## Option 1: GitHub Actions (Free)
If you export your project to GitHub, you can use a "Workflow" to build the APK automatically.

1.  **Export to GitHub**: Use the AI Studio settings to export your project.
2.  **Create Workflow**: In your GitHub repo, create a file at `.github/workflows/build_apk.yml`.
3.  **Paste this code** (⚠️ **STOP!** DO NOT copy the triple backticks ` ``` ` or the word `yaml`. ONLY copy starting from the word `name:`):

```yaml
name: Build Android APK
on:
  push:
    branches: [ main, master ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout Code
        uses: actions/checkout@v4
      
      - name: Install Java
        uses: actions/setup-java@v4
        with:
          distribution: 'temurin'
          java-version: '17'
          
      - name: Install Flutter
        uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.24.0'
          channel: 'stable'
          
      - name: Build Apps
        run: |
          # Build Child App
          cd flutter_child_app
          flutter pub get
          flutter build apk --release
          cd ..
          
          # Build Parent App
          cd flutter_parent_app
          flutter pub get
          flutter build apk --release
          cd ..
          
      - name: Upload Artifacts
        uses: actions/upload-artifact@v4
        with:
          name: apks
          path: |
            flutter_child_app/build/app/outputs/flutter-apk/app-release.apk
            flutter_parent_app/build/app/outputs/flutter-apk/app-release.apk
```


4.  **Get the file**: After you push the code, go to the "Actions" tab in GitHub. Once the build finishes, you can download the APK from the "Artifacts" section.

## Option 2: Codemagic (Easiest for Flutter)
Codemagic is a dedicated service for Flutter apps.

1.  Go to [Codemagic.io](https://codemagic.io).
2.  Connect your GitHub/GitLab account.
3.  Select your KiteControl project.
4.  Choose **Android** as the platform.
5.  Click **Start your first build**.
6.  Once done, they will email you the `.apk` file directly.

## Option 3: Command Line Only (Local)
If you have a terminal but don't want the heavy Android Studio UI:
1.  **Install Flutter SDK**: Follow the [official guide](https://docs.flutter.dev/get-started/install).
2.  **Install Command Line Tools**: You only need the "Android SDK Command-line Tools" (available in the "Command line tools only" section of the Android Studio download page).
3.  **Run Build**: `flutter build apk --release`.
